# Online Exam Portal
The objective of this project is to create a web-based Online Examination System that can conduct exams remotely. The main goal of this system is to develop a web application that can significantly reduce the time taken for exams and result declarations. The project aims to offer an effective platform for students and teachers alike, allowing for exams to be taken online.

## Requirements
XAMPP Software
Apache server
Programming Language: PHP, HTML, Javascript
Database: MySQL

## Installation
Clone this repository to obtain the code.

Paste all the files in the htdocs folder

Import the project.sql to import all the necessary tables

## Usage

Access the website using (http://localhost/Online-Exam-Portal/)
